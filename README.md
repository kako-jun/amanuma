# amanuma

React + Vite + TypeScriptで作成した落ち物パズルゲーム

## 🎮 概要

シンプルで楽しい落ち物パズルゲームです。Phaser 3ゲームエンジンを使用して作られています。

## ✨ 特徴

- React 18 + Vite 6による高速な開発環境
- TypeScript 5による型安全なコード
- Phaser 3ゲームエンジン
- Tailwind CSS 4によるスタイリング
- GitHub Pagesへの自動デプロイ

## 🎯 操作方法

- ⬅️➡️ 左右矢印キー：ブロックの移動
- ⬆️ 上矢印キー：ブロックの回転
- ⬇️ 下矢印キー：ブロックの高速落下

## 🚀 開発

### 必要な環境

- Node.js 20以上
- npm

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## 🛠️ 技術スタック

- **フレームワーク**: React 18、Vite 6
- **言語**: TypeScript 5
- **ゲームエンジン**: Phaser 3
- **スタイリング**: Tailwind CSS 4
- **開発ツール**: Prettier、ESLint、Husky、lint-staged
